# 인터페이스 자동 생성기

마우스로 화면을 조립해서 웹 사이트 형태로 저장하는 프로그램입니다.

복잡한 코드를 직접 작성하지 않아도 왼쪽 팔레트에서 필요한 기능 블록을 고르고, 가운데 캔버스에 배치한 뒤, 미리보기로 확인하고 저장할 수 있습니다. 저장된 결과는 별도의 폴더 안에 만들어지며, 함께 생성되는 `start-site.cmd` 파일을 실행하면 바로 사이트를 열 수 있습니다.

## 이런 작업에 사용할 수 있습니다

- 간단한 서비스 소개 화면 만들기
- 입력 폼, 표, 차트, 탭, 사이드 메뉴 같은 기능을 조합하기
- 화면 배치를 눈으로 보면서 빠르게 수정하기
- 저장하기 전에 새 창에서 실제 사이트처럼 확인하기
- 프로젝트 이름을 정해서 바탕화면에 실행 가능한 사이트 폴더로 저장하기

## 실행 방법

1. 이 폴더에서 `start-site.cmd` 파일을 더블 클릭합니다.
2. 필요한 프로그램이 설치되어 있지 않으면 자동으로 설치를 진행합니다.
3. 설치가 끝나면 브라우저가 열리고 인터페이스 자동 생성기가 실행됩니다.

브라우저가 자동으로 열리지 않으면 주소창에 아래 주소를 입력합니다.

```text
http://127.0.0.1:5173
```

## 기본 사용 순서

1. 왼쪽 팔레트에서 필요한 기능 블록을 선택합니다.
2. 블록을 캔버스의 원하는 위치로 드래그합니다.
3. 점선 그리드에 맞춰 위치와 크기를 조절합니다.
4. 오른쪽 설정 영역에서 제목, 설명, 버튼 이름, 색상, 언어 등을 바꿉니다.
5. `미리 보기`를 눌러 새 창에서 실제 결과를 확인합니다.
6. 프로젝트 이름을 입력한 뒤 저장합니다.

## 캔버스 사용 방법

- 블록은 저장하기 전까지 계속 움직일 수 있습니다.
- 선택한 블록은 마우스로 위치를 바꿀 수 있습니다.
- 블록 크기는 그리드 칸에 맞춰 늘리거나 줄일 수 있습니다.
- 선택한 블록은 `Del` 키로 삭제할 수 있습니다.
- 그리드 크기는 입력이 아니라 버튼 클릭으로 조절합니다.

## 팔레트 기능

팔레트에는 화면을 구성하는 기본 기능들이 들어 있습니다.

- 첫 화면 안내: 방문자에게 보여줄 대표 문구와 버튼
- 상단 네비게이션: 배치된 다른 기능 블록으로 이동하는 메뉴
- 사이드 메뉴: 화면 왼쪽 또는 오른쪽에 두는 보조 메뉴
- 입력 폼: 사용자가 내용을 입력하고 제출하는 영역
- 데이터 테이블: 행 추가, 삭제, 정렬, CSV 내보내기가 가능한 표
- 데이터 차트: 숫자 데이터를 막대나 선 형태로 보여주는 시각화 영역
- 탭 패널: 여러 내용을 탭으로 나눠 보여주는 영역
- 확인 모달: 저장, 삭제, 배포 같은 중요한 행동 전에 확인하는 팝업 형태

## 네비게이션 사용 방식

네비게이션은 처음부터 고정 메뉴를 가지고 있지 않습니다.

캔버스에 다른 기능 블록을 추가하면 그 블록 이름을 기준으로 네비게이션 버튼이 자동으로 생깁니다. 예를 들어 입력 폼과 데이터 테이블을 배치하면 네비게이션 안에 각각의 이동 버튼이 표시됩니다.

## 미리보기

`미리 보기` 버튼을 누르면 새 창에서 현재 캔버스 상태를 실제 사이트처럼 확인할 수 있습니다.

이 화면에서 입력 폼, 테이블, 차트, 탭, 네비게이션 같은 기능이 저장 전에 제대로 동작하는지 확인할 수 있습니다. 새 창이 열리지 않으면 브라우저에서 팝업 차단을 해제한 뒤 다시 시도합니다.

## 저장 결과

저장하면 바탕화면에 프로젝트 이름으로 된 폴더가 만들어집니다.

저장된 폴더는 아래처럼 구성됩니다.

```text
프로젝트 이름/
  start-site.cmd
  output/
```

- `start-site.cmd`: 저장된 사이트를 바로 실행하는 파일
- `output/`: 실제 사이트 파일이 들어 있는 폴더

일반 사용자는 저장된 폴더 안의 `start-site.cmd`를 더블 클릭하면 됩니다.

저장된 사이트는 저장할 때마다 실행 포트를 자동으로 새로 배정합니다. 그래서 여러 프로젝트를 저장해 두고 실행하더라도 같은 포트 번호 때문에 서로 충돌할 가능성이 줄어듭니다.

## 언어와 화면 설정

오른쪽 설정 영역에서 언어를 바꿀 수 있습니다.

한국어가 익숙한 사용자는 한국어로 화면을 보고, 다른 언어가 필요한 경우 선택한 언어에 맞춰 주요 안내 문구를 바꿀 수 있습니다. 테마 색상, 강조 색상, 배경 색상, 모서리 둥글기 같은 화면 분위기도 함께 조절할 수 있습니다.

## 문제가 생겼을 때

- 사이트가 열리지 않으면 `start-site.cmd`를 다시 실행합니다.
- 자동 생성기를 여는 주소는 기본적으로 `http://127.0.0.1:5173`입니다.
- 저장된 사이트는 저장할 때 자동으로 정해진 다른 포트로 열릴 수 있습니다.
- 미리보기 새 창이 열리지 않으면 브라우저 팝업 차단을 해제합니다.
- 저장이 되지 않으면 프로젝트 이름이 비어 있지 않은지 확인합니다.
- 처음 실행이 오래 걸리면 필요한 파일을 설치하는 중일 수 있습니다.

## 참고

이 프로그램은 코드를 몰라도 화면을 만들 수 있도록 설계되어 있습니다. 다만 저장된 결과물은 실제 웹 사이트 파일로 만들어지기 때문에, 나중에 개발자가 이어서 수정하거나 배포할 수도 있습니다.
